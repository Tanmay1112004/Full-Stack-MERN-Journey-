üè∑Ô∏è 3. HTML Attributes for Integration

id Attribute

Purpose: Unique identifier for sections

<!-- CORRECT --> <div id="sectionHome">Home</div> <div id="sectionFavouritePlaces">Favorites</div> <div id="sectionDetailedView">Details</div> <!-- WILL NOT WORK --> <div id="homeSection">Home</div> <!-- Wrong prefix --> <div id="favorites">Favorites</div> <!-- Missing prefix --> <div id="SectionDetails">Details</div> <!-- Capital S -->
onclick Attribute

Purpose: Handle click events for navigation

<!-- CORRECT SYNTAX --> <button onclick="display('sectionAbout')"> Go to About </button> <!-- INCORRECT SYNTAX --> <button onclick=display('sectionAbout')></button> <button onclick="display("sectionAbout")"></button> <button onclick="display(sectionAbout)"></button> <!-- RULE: Double quotes outside, single quotes inside -->

üîó 4. Step-by-Step Integration Process

Step 1: Prepare Individual Sections

Develop each section separately with complete HTML and CSS

<!-- Home Section --> <div class="home-container"> <h1>Tourism</h1> <button>Get Started</button> </div> <style> .home-container { background-image: url('ocean.jpg'); height: 100vh; } </style>
Step 2: Add Section IDs

Wrap each section in a div with ID starting with section

<!-- Add ID with 'section' prefix --> <div id="sectionHome"> <div class="home-container"> <h1>Tourism</h1> <button onclick="display('sectionFavouritePlaces')"> Get Started </button> </div> </div>
Step 3: Add Navigation Buttons

Add onclick attributes to buttons for navigation between sections

<!-- Navigation from Home to Favorites --> <button onclick="display('sectionFavouritePlaces')"> Get Started </button> <!-- Navigation from Favorites to Home --> <button onclick="display('sectionHome')"> Back </button>
Step 4: Add All Sections to Single File

Combine all sections in one HTML file with all CSS styles

<!-- Complete Structure --> <div id="sectionHome">...</div> <div id="sectionFavouritePlaces">...</div> <div id="sectionDetailedView">...</div> <!-- Add CCBP UI Kit Script --> <script src="ccbp-ui-kit.js"></script>
Step 5: Test Navigation

Click through all buttons to ensure proper section transitions

‚ö†Ô∏è Common Integration Mistakes

‚ùå Incorrect Section IDs:
  • id="home" - Missing 'section' prefix
  • id="section home" - Space in ID
  • id="SectionHome" - Capital 'S'
‚ùå Incorrect onclick Syntax:
  • onclick=display('section') - Missing quotes
  • onclick="display("section")" - Quote mismatch
  • onclick="display(section)" - Missing quotes inside
‚úÖ Correct Patterns:
<!-- CORRECT --> <div id="sectionHome">...</div> <button onclick="display('sectionAbout')">Go</button> <!-- ALWAYS: --> <!-- 1. section prefix in IDs --> <!-- 2. Double quotes for onclick --> <!-- 3. Single quotes inside display() -->
