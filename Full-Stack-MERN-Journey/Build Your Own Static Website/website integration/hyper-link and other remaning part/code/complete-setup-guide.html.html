üì∑ Image Hosting with Cloudinary

Cloudinary provides free image hosting with automatic optimization and CDN delivery.

Steps to Get Started:
  1. Go to cloudinary.com
  2. Click "Sign Up For Free"
  3. Fill in your details and verify email
  4. Go to "Media Library"
  5. Click "Upload" to upload your images
  6. Click any image to get its URL
Example Usage:
<!-- Using Cloudinary URL in HTML --> <img src="https://res.cloudinary.com/demo/image/upload/v1234567/sample.jpg" alt="Sample Image" class="img-fluid"> <!-- Using Cloudinary URL in CSS --> <style> .hero-section { background-image: url('https://res.cloudinary.com/demo/image/upload/v1234567/background.jpg'); background-size: cover; } </style>

Different Path Types:
<!-- Relative Paths --> <link rel="stylesheet" href="style.css"> <!-- Same folder --> <link rel="stylesheet" href="css/style.css"> <!-- In css folder --> <link rel="stylesheet" href="../styles.css"> <!-- One folder up --> <!-- Absolute Paths --> <link rel="stylesheet" href="/css/style.css"> <!-- From root --> <link rel="stylesheet" href="C:/project/style.css"> <!-- Full path --> <!-- External URLs --> <link rel="stylesheet" href="https://cdn.example.com/style.css">

üìù Quick Quiz

Question 1: Which HTML element is used to link HTML and CSS files?
Answer: <link> element

Example: <link rel="stylesheet" href="style.css">

Question 2: The HTML <link> element is a void element. True or False?
Answer: True

Void elements don't have closing tags. They are self-contained.

Question 3: Where should CSS links be placed in HTML?
Answer: Inside the <head> section

CSS files should be linked in the head for proper page rendering.
