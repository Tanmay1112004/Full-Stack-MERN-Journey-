Flexbox Quiz Answers

1. Which Bootstrap class name defines a Flexbox Container?
Answer: d-flex

Explanation: The class d-flex creates a flex container in Bootstrap.

2. What are the direct HTML elements in the Flexbox Container called?
Answer: flex items

Explanation: Direct children of a flex container are called flex items.

3. Identify the flex items in this code:
<div class="d-flex"> <div> <h1>Tourism</h1> <p>Plan your trip.</p> </div> </div>
Answer: div

Explanation: Only the <div> is a direct child (flex item). <h1> and <p> are not direct children.

4. Which Bootstrap class moves flex items horizontally?
Answer: flex-row

Explanation: flex-row arranges items horizontally (default direction).

5. Which Bootstrap class moves flex items vertically?
Answer: flex-column

Explanation: flex-column arranges items vertically from top to bottom.

6. The Bootstrap class name flex-row is the default Flex Direction for the Flexbox Container.
Answer: True

Explanation: When you use d-flex without specifying direction, flex-row is applied by default.

7. Which Bootstrap class aligns flex items at the start?
Answer: justify-content-start

Explanation: justify-content-start aligns items to the start of the container.

8. Which Bootstrap class aligns flex items at the center?
Answer: justify-content-center

Explanation: justify-content-center centers items along the main axis.

9. Which Bootstrap class provides equal space between flex items?
Answer: justify-content-between

Explanation: justify-content-between distributes items with equal space between them.

Flexbox Quick Reference

Container Classes:
  • d-flex - Creates flex container
  • flex-row - Horizontal layout (default)
  • flex-column - Vertical layout
Alignment Classes:
  • justify-content-start - Align to start
  • justify-content-center - Center alignment
  • justify-content-end - Align to end
  • justify-content-between - Equal space between
Key Concept: Flex items are only DIRECT children of the container with d-flex.
