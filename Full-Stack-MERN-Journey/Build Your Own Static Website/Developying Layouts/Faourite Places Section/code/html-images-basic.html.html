HTML Elements & CSS Box Model

1. HTML Image Element

Basic Syntax
<img 
  src="IMAGE_URL" 
  alt="Description" 
  class="class-name"
  width="300" 
  height="200"
/>

2. Void Elements

Void elements have only a start tag and no end tag.

Image:
<img src="photo.jpg" alt="Photo" />
Line Break:
<br />
Input:
<input type="text" name="username" />
Meta Tag:
<meta charset="UTF-8" />
Note: Void elements are self-closing. The forward slash (/) before the closing bracket is optional in HTML5 but good practice.

3. CSS Margin Property

Margin Visual Demonstration:

margin-top: 20px
margin-right: 20px
margin-bottom: 20px
margin-left: 20px
Element Content
padding: 30px
margin: 20px

Margin Examples:

margin: 10px
Equal margin on all sides
margin: 10px 20px
10px top/bottom, 20px right/left
margin: 10px 20px 30px
10px top, 20px right/left, 30px bottom
margin: 10px 20px 30px 40px
10px top, 20px right, 30px bottom, 40px left
Margin vs Padding
With Margin:
Space is OUTSIDE the border
With Padding:
Space is INSIDE the border
