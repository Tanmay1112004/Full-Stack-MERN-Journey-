Class 9 Quiz Answers

HTML Elements & CSS Box Model

1
Which HTML element defines an image?
Answer: img
Explanation: The <img> element is used to embed images in HTML documents.
<img src="photo.jpg" alt="Description" />
2
Which HTML Attribute specifies the path (URL) of the Image?
Answer: src
Explanation: The src attribute specifies the source file/location of the image.
Note: Always include the alt attribute for accessibility and SEO.
3
Which of the following is the correct syntax for defining an image?
Answer: <img src="image.jpg" />
Explanation: The correct syntax includes the <img> tag with src attribute.
Common Mistakes:
  • <image src="photo.jpg"> - Wrong element name
  • <img>photo.jpg</img> - Images don't have closing tags
  • <img source="photo.jpg" /> - Wrong attribute name
4
The image element in HTML is a void element and does not have closing tag
Answer: True
Explanation: Yes, <img> is a void element. Void elements only have a start tag and no end tag.
<!-- Void Elements --> <img src="image.jpg" /> <br /> <hr /> <input type="text" /> <meta charset="UTF-8" />
5
Which of the following is the correct syntax of a void element?
Answer: <br />
Explanation: <br /> is a void element for line break. The forward slash before closing is optional in HTML5 but recommended.
6
Identify the CSS property used to add space between elements.
Answer: margin
Explanation: margin adds space outside an element, while padding adds space inside.

7
Fill in the blank with an appropriate value for the given CSS Property: .card { margin: _____________; }
Answer: 20px
Explanation: CSS values don't need quotes. The correct format is 20px (not "20px" or 20 px with space).
/* Correct */ .card { margin: 20px; padding: 15px; border: 1px solid #ddd; } /* Incorrect */ .card { margin: "20px"; /* Quotes not needed */ margin: 20 px; /* No space between number and unit */ }
8
Which of the following CSS Properties will apply margin to the top of an HTML element?
Answer: margin-top
Explanation: CSS provides specific properties for each side:
.element { margin-top: 10px; /* Top margin */ margin-right: 15px; /* Right margin */ margin-bottom: 20px; /* Bottom margin */ margin-left: 5px; /* Left margin */ /* Shorthand for all sides */ margin: 10px 15px 20px 5px; /* top, right, bottom, left */ }

Key Takeaways

HTML Images:
  • Use <img> element
  • src attribute for image source
  • alt attribute for accessibility
  • Images are void elements (no closing tag)
CSS Margin:
  • margin creates space outside elements
  • Use specific properties: margin-top, etc.
  • Shorthand: margin: top right bottom left;
  • Values: numbers with units (e.g., 20px)
Pro Tip: Always use object-fit: cover for images to maintain aspect ratio while filling container.
