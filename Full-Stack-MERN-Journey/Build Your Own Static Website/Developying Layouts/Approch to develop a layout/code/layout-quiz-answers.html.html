4. What is the purpose of padding in CSS?

ðŸ“Œ Correct Answer: To specify the space around the four sides of the content of an HTML element (space between content and border).

ðŸ’¡ Key Points about Padding:
  • Creates space inside the element
  • Affected by background color
  • Useful for button click areas, card content spacing
  • Can be set individually for top, right, bottom, left
/* Button with good padding for better UX */ .button { padding: 12px 24px; /* Vertical | Horizontal */ margin: 10px; /* Space between buttons */ background-color: #007bff; color: white; border: none; border-radius: 5px; } /* Individual padding */ .card { padding-top: 20px; padding-right: 15px; padding-bottom: 20px; padding-left: 15px; /* Shorthand: padding: 20px 15px; */ }
