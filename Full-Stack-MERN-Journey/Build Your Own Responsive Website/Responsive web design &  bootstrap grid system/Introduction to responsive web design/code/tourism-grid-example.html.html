The Bootstrap Grid System is a mobile-first, responsive grid system that uses containers, rows, and columns to layout and align content. It's built with flexbox and allows up to 12 columns across the page.

5. Grid System Rules

üìå Rule 1: Containers contain Rows

Rows must be placed inside a container for proper padding and alignment.

üìå Rule 2: Rows contain Columns

Columns should always be placed inside rows to maintain proper gutter spacing.

üìå Rule 3: Columns total should be ‚â§ 12

The sum of column numbers in a row should not exceed 12. Extra columns will wrap to a new line.

üìå Rule 4: Use content inside columns

Only place content inside columns. Never place content directly in rows or containers.

üìù Quick Quiz

Question 1: How many columns does Bootstrap Grid System have?
Answer: 12 columns

Bootstrap uses a 12-column fluid grid system that scales with the device.

Question 2: What is the correct order of grid components?
Answer: Container ‚Üí Row ‚Üí Column

Containers hold rows, rows hold columns, and columns hold content.

Question 3: What happens if column numbers exceed 12 in a row?
Answer: Extra columns wrap to a new line

Columns exceeding 12 will automatically wrap to the next line.

Question 4: What class creates a full-width column?
Answer: col-12

col-12 occupies all 12 columns, making it full width.
