Code Implementation

Complete Example Code:

<!-- Bootstrap Grid with Colors and Spacing -->
<div class="container">
    <div class="row">
        <!-- Responsive columns -->
        <div class="col-12 col-sm-6 col-md-4 col-lg-3 col-xl-2 mb-3">
            <div class="bg-primary p-3 text-white">
                <h5>Primary</h5>
                <p>#007bff</p>
            </div>
        </div>
        
        <div class="col-12 col-sm-6 col-md-4 col-lg-3 col-xl-2 mb-3">
            <div class="bg-secondary p-3 text-white">
                <h5>Secondary</h5>
                <p>#6c757d</p>
            </div>
        </div>
        
        <!-- More colors... -->
    </div>
    
    <!-- Cards with spacing -->
    <div class="row mt-4">
        <div class="col-12 col-md-6 col-lg-4 mb-3">
            <div class="card p-3">
                <h5>Card Title</h5>
                <p>Card content</p>
                <button class="btn btn-primary px-4">
                    Action
                </button>
            </div>
        </div>
    </div>
</div>

Mastering spacing utilities, background colors, and responsive layouts.

Key Takeaways:
  • Use m-* and p-* for spacing
  • Never use CSS margins on grid columns
  • Combine with responsive breakpoints
  • Use bg-* classes for backgrounds
