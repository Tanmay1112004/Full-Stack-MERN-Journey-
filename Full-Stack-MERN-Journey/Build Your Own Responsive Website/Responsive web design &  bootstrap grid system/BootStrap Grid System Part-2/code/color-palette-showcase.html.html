Spacing Utilities

Margin Utilities
Size Scale:
  • 0 = 0px
  • 1 = 4px (0.25rem)
  • 2 = 8px (0.5rem)
  • 3 = 16px (1rem)
  • 4 = 24px (1.5rem)
  • 5 = 48px (3rem)
Examples:
mb-1 (4px bottom margin)
mb-2 (8px bottom margin)
mb-3 (16px bottom margin)
mb-4 (24px bottom margin)
mb-5 (48px bottom margin)
Padding Utilities
Same Size Scale:
  • 0 = 0px
  • 1 = 4px (0.25rem)
  • 2 = 8px (0.5rem)
  • 3 = 16px (1rem)
  • 4 = 24px (1.5rem)
  • 5 = 48px (3rem)
Examples:
p-1 (4px padding)
p-2 (8px padding)
p-3 (16px padding)
p-4 (24px padding)
p-5 (48px padding)
Responsive Spacing

Spacing utilities can be combined with responsive breakpoints:

<!-- Different spacing on different screens --> <div class="p-2 p-md-3 p-lg-4"> Content with responsive padding </div> <!-- Mobile: mb-3, Desktop: mb-4 --> <div class="mb-3 mb-md-4"> Responsive margin bottom </div> <!-- Mobile: p-2, Tablet: p-3, Desktop: p-4 --> <div class="p-2 p-sm-3 p-md-4"> Progressive padding </div>
p-2 p-md-3 p-lg-4 (Responsive padding)
mb-3 mb-md-4 mb-lg-5 (Responsive margin)

Card Component

Bootstrap Utilities

Master Bootstrap's spacing and color utilities for professional web development.

Key Utilities
  • m-* - Margin
  • p-* - Padding
  • bg-* - Background
  • text-* - Text color
Remember

Never use CSS margins on grid columns! Use Bootstrap's spacing utilities instead.
