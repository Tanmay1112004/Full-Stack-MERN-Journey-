Column Wrapping & Responsive Breakpoints

1. Column Wrapping

When the total width of columns in a row exceeds 12, extra columns wrap to a new line.

Example: 4 columns of col-6

<div class="row"> <div class="col-6">Column 1 (6)</div> <div class="col-6">Column 2 (6)</div> <!-- Total: 12 --> <div class="col-6">Column 3 (6)</div> <!-- WRAPS: New line --> <div class="col-6">Column 4 (6)</div> <!-- Total: 12 in new line --> </div>

4. Inheritance Principle

In Bootstrap's mobile-first approach, larger screens inherit smaller screen behavior unless overridden.

üìñ Rule of Inheritance:

If you define a behavior for a smaller screen, it applies to all larger screens unless you specify different behavior for those larger screens.

<!-- Example 1: Mobile behavior inherited --> <div class="col-12"> This is full width on ALL devices </div> <!-- Example 2: Mobile behavior overridden --> <div class="col-12 col-md-6"> Mobile: Full width (col-12)

üìù Quick Quiz

Question 1: What happens when columns exceed 12 in a row?
Answer: Extra columns wrap to a new line

The grid system automatically moves exceeding columns to the next row.

Question 2: What's the class prefix for tablets (‚â•768px)?
Answer: col-md-

md stands for medium devices (tablets).

Question 3: Does Bootstrap use mobile-first or desktop-first approach?
Answer: Mobile-first approach

Design for mobile first, then enhance for larger screens.

Question 4: What will this column do: <div class="col-6 col-md-4">?
Answer: Mobile: 6 columns, Tablet+: 4 columns

It takes half width on mobile and one-third width on tablets and larger.
